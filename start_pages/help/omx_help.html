Introduction

OMX is a video player designed to run through the command line.
While the player is natively without it's own controls, it runs video smoothly.

Requirements

Requirement Purpose
sh plugin Finding and starting video files

Raspberry Pi 2/3 server installation

Should work (and be installed) natively on Raspberry Pi.
If not it can be installed using sudo apt-get install omxplayer

Plugin Configuration

settings.properties

1. Add omx to plugins > active
2. Add the extensions of video files to be opened with OMX to omx > omx.extensions
3. Make sure directories > videos contains all the folders you want to look for video files
4. Add lxterminal -e omxplayer to shell > allowed.commands
