Raspberry Pi 2/3 server installation

A standard installation of VLC on a Raspberry Pi suffers from a lack of CPU power.
This means VLC has a hard time displaying any video with the normal executable.
Solving this is a matter of compiling and installing VLC manually, including hardware acceleration.

Guide for installation of VLC with HW acceleration
https://www.raspberrypi.org/forums/viewtopic.php?f=66&t=59814

Plugin Configuration

Configuring the preferences of VLC itself is done manually.

settings.properties

1. Add vlc to plugins > active
2. Add <path to vlc executable> to vlc > executable.paths
3. Add the extensions of video files to be opened with VLC to vlc > extensions
4. Make sure directories > videos contains all the folders you want to look for video files
